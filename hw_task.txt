 1) Create an external repository named JSON:
	In GitHub account  click on the Your repositories -> New -> write Repository name -> Creare repository

 2) Clone the JSON repository to the local computer:
	- Click on the Code button and copy HTTPS
	- In the Terminal type git clone and add HTTPS

 3) Create a "new.json" file inside the local JSON
	touch new.json

 4) Add the file under the git
	git add . "new.json"

 5) Commit the file
	git commit -m "Message"

 6) Send the file to an external GitHub repository
	git push

 7) Edit the content of the file "new.json" - write information about yourself (name, age, number of pets, future desired salary).
    Write everything in JSON format
	vim new.json

 8) Send the changes to an external repository
	git status
	git commit -am "Message"
	git push

 9) Create a "preferences.json" file
	touch preferences.json

 10) In the "preferences.json" file, add information about your preferences (favorite movie, favorite show, favorite food, favorite time of year,
     party yo would like to visit) in JSON format
	vim prefereces.json

 11) Create a "skills.json" file to add information about skills that will be studied on the course in JSON format
	touch skills.json
	vim skills.json

 12) Send 2 files to an external repository at once
	git status
	git add .
	git push

 13) Create the "bug_report.json" file on the web interface
	- In GitHub account click Add file -> Create new file -> add Commit new file -> Commit new file

 14) Make Commit changes on the web interface
	git fetch
	git pull

 15) On the web interface  modify the "bug_report.json" file and add the bug report in JSON format.
	
 16) Make Commit changes (save) the changes in the web interface.

 17) Synchronize external and local JSON repository
	git fetch
	git pull
